General Theme of Change

  • Gameplay Changes
  • ALICE Changes
  • Visual Changes
  • Bug Fixes

Following up on the last version, this is another small incremental release. There are a few difficulty adjustments to Hell, including a new boss variation and some minor spawn adjustments. A new experimental boss spawning system is now in place to prevent spawns too close to players, it may require some adjustments in the next few releases. This version also includes an automatic trigger safety system to prevent privates and leavers from attacking allies. Some ability and item icons have been updated to be HD or replaced entirely. Lastly, a few long standing bugs (and some very recent ones) have been fixed.

Changes

Gameplay Changes

  • The timer for securing Ravenholm in Lost Cause is now based on difficulty (30s, 45s, 60s)
  • You can now replace the strobe for securing Ravenholm in Lost Cause the moment it is destroyed.
  • The Laser Designator no longer overrides the camera quick position hotkey.
  • Everyone can now change the number of fire mission shells if the player owning the FO or IUL leaves the game.
  • Increased airfield communications tower activation radius from 325 to 450.
  • Marine Command Authorization for Airstrikes is now given on Containment to align with the other variations of chapter 3.
  • Garg waves will stop spawning when the airfield tower is destroyed in chapter 2.
  • M320 will now have priority taking charges over M29 and Mk17 when picking up HEDP ammo.
  • CWO Nadav's helicopter in Containment can now evacuate casualties.
  • Added 3 new pet types.
  • Hellfire upgrades no longer spawn if there's no MQ-3 Predator in the game.
  • Zulu bosses can no longer spawn too close to players.
  • Dark clouds can no longer reappear if they showed up at the start of chapter 1 and then disappeared (hell difficulty only).
  • Added new version of Eris.
  • The EM Sphere now deals reduced damage (about 60%) in hell difficulty.
  • Added chance of Laufey spawn in Magnificent Bastards.
  • Attempting to fire the FGM-148 Javelin while too far away or too close to the target will no longer trigger cooldown.
  • The FGM-148 Javelin can no longer be fired if the user doesn't have vision of the target.
  • Taking control of a helicopter with a marine pilot will now increase zulu hp (same way as when marine reinforcements arrive).

ALICE Changes

  • Added automatic trigger safety lock mechanism to prevent friendly fire when players are rank Lance Corporal or below.
  • Trigger safety lock is also always enabled for units of player that have left the game.
  • Changed tagging system ping color for bosses to be different from the player hit alert.

Visual changes

  • Updated a few icons to HD.
  • Decreased the Eris encounter RGB filter red component from 60 to 40.
  • Changed the effect for Fire Missions / TOT missions / WP to be artillery shells instead of the infernal spawn effect.
  • Added info in the Javelin tooltip to indicate that it can only be used by the Assaultman.
  • The Javelin impact explosion now lasts until the smoke clears naturally, instead of disappearing immediately.
  • [Delta Operator]: Updated tooltip of Covert Operations to prevent text from being cut in some screen resolutions.
  • [Infantry Unit Leader]: The Defensive Stance buff is now shown as a buff instead of a debuff.

Bug Fixes

  • The idle worker icon is no longer shown while using the FGM-148 Javelin.
  • Fixed FGM-148 Javelin not having the correct model when being air dropped.
  • Fixed FGM-148 Javelin items not being dropped when the carrier dies.
  • The minimum value for changing acquisiton range is now 51 instead of 50.
  • The pet Play Dead ability can no longer be used while jumping.
  • Units that were being repaired on the airfield bay while chapter changed no longer get constant repair.
  • USS Independence can no longer supply ammo above the limit.
  • Zulus will no longer aggro on the airfield tower after Chapter 2 airfield defense is over.
  • Golf explosives, vehicles and marines no longer spawn ghouls in Lost Cause.
  • Explosives are no longer valid repair targets on the Airfield repair bay.
  • Vehicles no longer use player ammunition.
  • Experience loss (through gameplay or MOS change) is now saved on normal difficulty games for players ranked Colonel and above.
  • Zulu bosses can no longer gain experience killing players.
  • Fix field light effect disappearing when dropping Mk17 while holding the weapon.
  • Marine Command Authorization for Landing Beacons, Airstrikes, Gunship Support and Fire Missions is now also given for leavers.
  • The containment helicopter should no longer move before refueling.
  • Fixed unit movement speed increasing permanently when picking up a second javelin item.
