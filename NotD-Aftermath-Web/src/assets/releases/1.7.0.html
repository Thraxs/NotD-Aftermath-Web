General Theme of Change

This is the first official release since 1.6a, keep in mind that you may have seen most of the changes in previous experimental or beta versions.

Changes

Gameplay Changes

  • Added Hell difficulty.
  • Immortals killed with TX-1 ammunition now have a lower chance of resurrection.
  • Increased the number of LZ names on Magnificent Bastards to 12.
  • You can now save when you get more than 3000 experience in a game. The limit is still in place, so only 3000 will be saved if you go above it.
  • Pilot is now available as a reinforcement on Chapter 1.
  • Updated DAO item description to reflect damage increase.
  • Reduced Monofilament Cartridge stun duration from 2s to 1.5s for Marksman minis.
  • Added Field Marshal rank at 1M exp. +1 damage, +0.02 attack speed, +0.03 energy regeneration, +2 move speed, -7HP, +15 sight range. Can order hold position to all ranks. Double chance of pilot reinforcement. Extra skill point on spawn.
  • Unknown entities spawned from Prometheus corpse are now timed life. Parasites are still the same.
  • Added new watchtower in the SW entrance of the airfield. Slightly adjusted pathing for all towers.
  • Added FN Mk17 (SCAR-H) battle rifle for Rifleman/Infantry Unit Leader.
  • You can no longer get a fracture while jumping (to prevent instant death)
  • Extra 50 XP to all Marines when all 16 civilians make it through the Convoy Mission. 75 XP on Nightmare. 100xp on Hell.
  • Field combat rank experience is now shared map wide.
  • C-130 item drops now deal damage the moment they touch the ground, not before.
  • Chain explosives can now be stacked up to 9 without Ordnance Handling to match default stack size of 3.
  • The current saving algorithm doesn't allow encoding experience values above 1M, so saving is now limited automatically to 1M. This should fix bugged codes when you have more experience than this.
  • Civilians will now drop inventory items when boarding USS Independence.
  • Marine Pilots can no longer pick up heavy weapons and special ammunition.
  • Removed inventory from USS Independence.
  • The cobras chasing the Flesh Composite during Chapter 1 no longer have hellfire missiles.
  • Chain explosives now deal extra damage if detonated by claymores.
  • Reloading manually is now slightly faster than reloading automatically.
  • Added buff icon for immortal resurrection damage immunity.
  • Immortals can now be killed by claymores and chain explosives.
  • Added message when reloading pistol.
  • Diablo can no longer spawn when there are less than 4 players alive.
  • Decreased Parasite mounds base damage from 35 to 15. But they will now inflict disease on every hit and will never inflict any other ailment.
  • Changed the mechanic for securing Ravenholm during Lost Cause. The mission is now placing and protecting an infrared strobe for 30 seconds.
  • [MQ3 Predator]: New ability "FLIR sector scan". Unlocked by Hellfire Upgrade. Performs a thermal scan of Zulus on the sector.
  • [Designated Marksman]: Reduced stun duration of Monofilament Cartridge to half and added slow effect.
  • [Assaultman]: Updated text and icon for MD5 stun effect. It was previously the default impale effect.
  • [Assaultman]: Increased starting energy from 100 to 150.
  • [Assaultman]: Added SMAW AoE reticle.
  • [Assaultman]: Added FGM-148 Javelin missile launcher item.
  • [Forward Observer]: Added ALICE notification when Perimeter Intelligence is activated.
  • [Forward Observer]: Remove Transceiver ability.
  • [Forward Observer]: Added damage dealt information to Orbital Artillery tooltip.
  • [Forward Observer]: White phosphorus now reduces the armor of the units hit by the initial explosions for a few seconds.
  • [Combat Engineer]: Reduced number of levels of Critical Strike from 7 to 5, increased the damage of each level. Now insert shells is granted at level 2 instead of level 3.
  • [Infantry Unit Leader]: Switched hotkey for defensive stance to "S".
  • [Combat Engineer]: Updated projectile speed from 2250 to 2500.
  • [Delta Operator]: The CM bonus armor aura dummy is now invisible and unclickable.

ALICE Changes

  • Updated ALICE Operations Manual to include view '3' and '4'.
  • Team colors are now disabled by default on survival (no need to type -tc -1 anymore). No change for deathmatch.
  • ACA is now disabled by default. An ALICE message will be shown alerting that ACA is available for privates if they want to enable it.
  • The "-team" command now displays the classes for KIA marines.
  • ALICE will now display a message to all players when the M242 is disassembled.
  • The "-report" command now displays the squad experience factor. This value is a percentage where 100% is the initial factor for normal difficulty with 0 squad rating. It can increase or decrease depending on many factors throughout the game. The higher it is, the more rank experience the squad will earn.
  • Increased degree of randomness based on user input for the "-random" command.
  • Added new "-ar x" command to set the acquisition range of your hero between 50 and the max attack range.
  • Added new "-uar x" command to set the acquisition range of your selected unit between 50 and the max attack range.
  • Added new "-norf" command to opt out of reinforcements for players with rank of MajGen and above.
  • Shell adjustment messages are now only shown to the player doing them.
  • Added context tagging system for communicating locations to allies.
  • Added new "-kick x" command to kick a player (red player only).

  • Added new icon, updated name and description for Lost Cause enhanced energy regeneration buff.
  • Added new custom icon for the detonation cord item.

  • Remove extra line break in CE net skill description.

Bug Fixes

  • Fixed intro music not playing.
  • The enhanced energy regeneration ability granted during Lost Cause was taking up an ability slot. It will now be always hidden.
  • You will no longer lose extra energy cells if you have more than required when entering power generator during Nightstalkers.
  • Lieutenant Oliver can no longer trigger fire missions after he dies during Lost Cause.
  • Marines that spawn on the Containment helicopter are now controled by the AI instead of the dark green player.
  • Fixed immortal AI to make them attack the unit that killed them immediately on resurrection.
  • Updated Lost Cause golf helicopter spawn message from Iron Hand. It no longer says "Patching in its last known coordinates".
  • Orbital Artillery shock is no longer permanent.
  • Refactored weather effect caching. This should fix effects like BHD wind not disappearing after a while like they are supposed to.
  • The "Attack Formation" sound will no longer play for the Lyganesh F-22s after it has been called off.
  • Fixed color used in text commands for dark green player to be dark green instead of light blue.
  • Fixed positioning of Containment helicopter minis.
  • Added small wait between evacuating kilos and evacuating civs to avoid missed bodies.
  • The Lyganesh CAS mission on Revelation will no longer hit the east side of Ravenholm.
  • The survivors of Black Hawk Down can no longer be rescued from town.
  • Failing to rescue the Black Hawk Down will no longer result in extra spawns for Chapter 1.
  • Fixed stats for Drill Instructor Ribbon not being applied.
  • Reduced energy cell discharge bonus from 200 to 150 to match the tooltip.
  • Added failsafe to prevent giving moirae speed boost to other bosses.
  • Hades and Diablo will no longer snatch players with dead or unselected heroes.
  • Fixed incorrect hotkey text for Satellite Imaging ability (General) on communications module.
  • Fixed shotgun sound missing on intro C.
  • Using NVG or Sat Scan on the comms module no longer disables FPS mode.
  • Added failsafe to prevent the chapter 1 helicopters pursuing the Flesh Composite from spawning later in the game.
  • [Machine Gunner]: Ammunition link can no longer be used while jumping.

  • Removed functionality for loading codes from version 1.4b and earlier
  • Internal reorganization of core engine tools.

  • Small text changes to Quest Menu
