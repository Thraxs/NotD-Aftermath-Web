Another incremental release. There aren't too many changes this time, mostly a small rework of the Infantry Unit Leader abiltiies and some changes to the Forward Observer and the Flamethrower classes. All game breaking bugs from 1.7.2 should hopefully be fixed in this version.

Changes

Gameplay Changes

  • Restored EM Sphere damage to the value in 1.7.1 (no longer reduced in Hell difficulty)
  • Replaced Thanatos spawn during convoy in Hell difficulty with a new Etaphreven variant.
  • The convoy boss can no longer spawn right when poppers spawn.
  • Completing the missions for caves and spores will now increase the level of the Force Recon while keeping his leveling progress (in relative percentage).
  • Increased Medusa radiation range from 300 to 600.
  • Reduced the Hold Position armor bonus from 25 to 5.
  • [Flamethrower]: Removed Fortitudine ability.
  • [Flamethrower]: Added suit integrated blastocyst module ability. Adds HP regeneration and ailment resistance.
  • [Infantry Unit Leader]: Removed Fire Mission ability.
  • [Infantry Unit Leader]: Removed Saline IV ability.
  • [Infantry Unit Leader]: Removed Light Weapons Training ability.
  • [Infantry Unit Leader]: Added Fortitudine ability (without ailment resistance)
  • [Infantry Unit Leader]: Added Leadership skills ability. Leadership skills like the designator and the stances now require leveling this ability.
  • [Infantry Unit Leader]: Added Helicopter Pick Up ability.
  • [Infantry Unit Leader]: Added Heads Up! ability.
  • [Forward Observer]: Added MGRS Fire Mission ability that fires on coordinates given using a Military Grid Reference System. Unlimited range.
  • [Forward Observer]: Reduced starting shells from 900 to (600, 500, 400) on Normal, Nightmare and Hell respectively.
  • [Forward Observer]: Reduced Fire Mission range from 15000 to 6000.

ALICE Changes

  • Removed -uar command due to high desynchronization chance when getting player selections.
  • [Forward Observer]: Added -fm command to set grid coordinates for MGRS Fire Missions.
  • [Forward Observer]: Shell dispersion can now be changed by anyone if the player playing Forward Observer leaves the game.

Visual changes

  • Removed light effect from artillery shell model. Increased size by 25%.
  • Updated icon for fortitudine ability.
  • Updated icon for Request Gunship Support and helicopter units.
  • Updated the magazine icon.
  • Fixed typo in Lost Cause Iron Hand messages.
  • Removed Mundus aura effect.

Bug Fixes

  • Fixed secondary explosions missing from airstrikes, satchels and hellfires.
  • Fixed mission not progressing after killing Hades.
  • Fixed multiple instances of USS Independence spawning on Chapter 1.
  • [Flamethrower]: Getting ammunition from USS Independence when near max fuel will no longer result in fuel loss.
  • [Combat Engineer]: The communications array will no longer be placed if the cast is interrupted.

Other

  • Internal refactor of Waves 6 to 9 into different vJass scopes.
