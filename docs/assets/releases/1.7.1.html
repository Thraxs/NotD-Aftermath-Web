This is a small patch release with some balancing changes and bug fixes.

Changes

Gameplay Changes

  • Parasite Mound damage increased from 15 to 20. The disease effect is now single target instead of AoE.
  • You can now only carry 1 Javelin missile in your inventory (including empty missile tubes).
  • Decreased the radius of damage done by items when being airdropped from 250 to 150.
  • The Lyganesh CAS mission on Revelations can no longer happen after Chapter 2 ends.
  • Defeating Havok now gives an XP bonus.

ALICE Changes

  • The "-shells" command now only changes fire mission shells if used by a Forward Observer or Infantry Unit Leader.
  • Improvements to the "ctrl + click" tagging system target detection.

Visual changes

  • The scoreboard now displays simplified class icons instead of the HD icons for the heroes.
  • Added different model for the Ground Illumination Flare instead of using the same one as the Smoke Grenades.

Bug Fixes

  • Ghouls can no longer spawn from disease clouds.
  • Fixed typo in Javelin CLU description.
  • Fixed Javelin missile not homing on targets.
  • Fixed Javelin CLU being unable to target spell immune units.
  • Fixed explosive detonations not preventing immortal resurrection.
  • Updated Revelations Eris death event detection.
  • Updated "-kick" command to reduce chance of desync.
  • Marines will now reload when they reach 0 mana instead of attempting to shoot without ammo.
  • Disabled target tagging on placed ground illumination flares.
