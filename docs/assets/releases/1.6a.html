General Theme of Change

Changes

New Features and Improvements

  • Ranking system implemented for official hostbots.
  • Added "-view 4" for a closer up view to compensate for the widescreen changes.
  • Added supply drop animation for C-130 airdrop items.
  • Reverted Chapter 3 probabilities back to their original values in 1.5c.
  • Poppers will now always pop if they aren't killed within a certain amount of time (TH & Spinaker).
  • Improved loading times.
  • JASS optimizations.

Bug Fixes

  • Golf Company is no longer immune to zombie attacks in Lost Cause.
  • Fire Mission base damage updated from 300 to 250 in order to match the tooltip (Spinaker).
  • Fixed a bug that could cause the boss in MB to spawn as the wrong player.
  • Zombies are grey instead of black on the Minimap.
  • Fixed an issue where zombies or Bosses might not attack.
