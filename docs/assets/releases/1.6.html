General Theme of Change

  • Path 1.29 Compatibility
  • Features and Improvements
  • Bug Fixes

Changes

Patch 1.29 Compatibility

  • Removed multiple instances of the "return bug" in favor of a different typecasting method.
  • Replaced the human borders of the UI with the night elf ones for the extended widrescreen UI components.
  • Replaced all instances of Player(13) and Player(14) with bj_PLAYER_NEUTRAL_VICTIM and bj_PLAYER_NEUTRAL_EXTRA respectively. ConvertPlayerColor and bj_FORCE_PLAYER instances were updated as well.
  • Changed Residence 41 from Red to Brown.

New Features and Improvements

  • Save codes are now stored in your Documents\NOTD Aftermath\ folder.
  • Added "-autoload" command which will load your last save automatically without typing in your code.
  • Reverted the loading screen to the one in 1.5c.
  • Textures are now compatable with Mac.
  • Improved the compression of many textures making them look better while having a small filesize.
  • EM Sphere no longer releases if Residence 41 is dead.

Bug Fixes

  • You no longer have to type -view after completing a mission to reset your camera.
  • Improved the detection of when a unit is alive or not. It should be 100% accurate now. This has the potential to fix some random, rare bugs. Currently only applied to Chapter 1.
  • The "-clear" command no longer clears messages for all players (Spinaker).
  • Fixed a fatal error that would happen within the first few minutes of the game due to handle stack corruption. This may also fix more random crashes.
  • Fixed a bug in Revelations where you could get stuck in Chapter 2 if you didn't kill the bosses in the correct order (Spinaker).
  • Fixed a fatal error that could occur if Residence 41 died before the EM Sphere releases.
  • You can no longer play as the Marine Corp or the Civilians by using a hostbot.
  • Fire Mission base damage updated from 300 to 400 (Spinaker).
  • Fixed a bug where you could get stuck floating in the air while using Jump.
  • Detonating a MD5 Mine on a unit with Napalm no longer crashes the game.
